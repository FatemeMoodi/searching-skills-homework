What is margin collapse:

Margin collapse occurs when two elements with adjacent margins are placed next to each other. In this case, if the margins be equal, insted of adding the values of both margins together, just one value apply.
and if they weren't equal, the larger of the two margins is used as the spacing between the elements, rather than adding the values of both margins together. Margin collapse means that the distance between two elements with adjacent margins is equal to the larger margin of the two, instead of the sum of both margins. Generally, margins that are in the same direction and on the same line can be added together, but if two elements with adjacent margins are placed in different directions, their margins do not add up and the larger margin is used as the spacing between them. Margin collapse can cause problems in web page design, so it should be used carefully and alternative methods such as using padding instead of margin should be considered.
